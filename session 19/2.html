Process Explanations:

  • Adding: Form Submit → packaged into object → array.push() → trigger render() function to display.
  • Editing: Click Edit → Populate fields into Form → Modify items → Save back to specific array index → trigger render().
  • Deleting: Click Delete → array.splice(index, 1) → trigger render() to immediately update view.

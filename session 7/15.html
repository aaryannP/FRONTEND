DOM event flow standard order ko define karta hai jiske mutabik web page par kisi element par event hone par browser events ko handle karta hai (isiki phases Capturing, Target, aur Bubbling hoti hain). Flipkart app me, agar aap kisi product card ke andar bane 'Add to Cart' button par click karte hain, to event bubbling ke karan wo click event automatic upar parent container (poore product card) par bhi trigger ho jayega. Isse unexpected behavior ho sakta hai, jaise button click karne par product cart me add hone ke sath-sath poora product details page bhi galti se open ho jaye.
